# AceUp 🚀

### From Panic to Prepared

AceUp is an AI-powered exam preparation platform designed to help students reduce procrastination and study more effectively by centralizing all learning resources into one intelligent dashboard.

---

## 📌 Problem Statement

Students waste significant time searching through scattered PDFs, notes, syllabi, and previous year question papers (PYQs). This leads to procrastination, stress, and inefficient exam preparation.

---

## 💡 Solution

AceUp simplifies exam preparation by:

* Collecting and organizing study resources
* Allowing users to upload their own notes
* Generating AI-powered summaries, flashcards, and quizzes
* Highlighting important topics using "Panic Mode"

---

## ✨ Features

### 🔍 Smart Resource Search

Automatically discovers publicly available study materials like PYQs, syllabi, and notes.

### 📁 File Upload Support

Users can upload PDFs, images, and handwritten notes for personalized study material generation.

### 🚨 Panic Mode

A focused revision mode that highlights:

* High-weightage topics
* Frequently asked questions
* Key concepts for quick revision

### 🧠 AI-Powered Learning Tools

Generates:

* Summaries
* Flashcards
* Practice quizzes
* Important topic breakdowns
